>> >> Image Examples
 
Implementing images on xHTML is very basic and relevant, it brings more life to the page, rather then to be all plain text. Most of the pages has images, they can be clickable or not clickable. Images can be implemented to link the user to another relative page. One relative example of a xHTML image is the Advertisements on the web page. Pages such as Amazon, Bestbuy, Tigerdirect, they implement the image linking, by this mean, when the customer clicks on the image of the product they are looking for, they will be redirected to another page which has more specific information about the product. There are some common tags for images, which are border(providing the image a thin or thick border), alt, title, width, height, etc. Images can be resized, can be also aligned.
