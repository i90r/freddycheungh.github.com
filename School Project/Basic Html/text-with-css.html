>> >> Text Examples
 
On xHTML, text is the main and most important element, which people use on the creation of their own web pages. Through text, the creator expresses ideas to the public, without text, they will be no communication. Every web page has text on them, take as an example any web page, it can be just as simple as a brief welcome, or some other information that the creator wants the user to see such as a news tag, on CNN web page, a simple feed from facebook and so much more. There are many tags for text, some of the most common ones are <p>(paragraph), <b>(bold), <i>(italics), blockquote, font colours, and many others. Depending on the designer, several tags can be implemented to make a more user friendly paragraph.
