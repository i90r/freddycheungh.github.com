>> >> Table Examples
 
Tables in xHTML are very frequently used, specially for charts. Many pages such as shopping sites uses table for the size chart. There are also table for recepies, cellphone specifications, and many other things which gives the user an easier way to search what they want. There are many tags or attributes which can be used on tables such as the width, border, background, cellspacing, align, and so many mores. Just to name a few, they can change the size of the table, align the content of it and many other things making it more user friendly.
