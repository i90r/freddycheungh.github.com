>> >> List Examples
 
On xHTML pages, listing is one of the most common things to use. Many pages use list to display their options, such as video games pages, which allows the customer to read about the product they are interested in or the date which the product is going to be on sales. Not only game pages uses listing, other pages such as Wikipedia, Amazon, etc, allowing the user to see the information is a easier and neater way. There are also tags for listing allowing them to be ordered by numbers, letters, roman numbers, squares, filled circles and empty circles. Also, there are detailed listing, which are less frequent on xHTML. There is also a nested list, which will show a more complex listing with more information and can be more precise.
